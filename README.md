# Tutorials SPA built on top of Vue.js, Axios and Tailwind CSS (it consumes the REST API: tutorials-api)

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
